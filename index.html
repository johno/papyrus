Welcome to Papyrus

A minimalist browser built for readability

Why papyrus?

Because the web is becoming unreadable. Papyrus seeks to render the web in Safari's reader mode all of the time.

How does it work?

Papyrus removes all styling from a page and then intelligently adds new CSS based on content. This is currently extremely alpha and will likely break a lot of sites. It also doesn't currently handle SPAs because it only does its work on initial render.
